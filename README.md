# NocoBase Documentation

## Development

```bash
git clone https://github.com/nocobase/docs.git
cd docs
yarn install
# 运行中文文档
yarn dev
# or run en-US document
yarn dev:en
# ou lancer fr-FR document
yarn dev:fr
# または、日本語のドキュメントを実行する
yarn dev:ja
```

Note: `yarn dev` and `yarn dev:en` cannot be run at the same time
