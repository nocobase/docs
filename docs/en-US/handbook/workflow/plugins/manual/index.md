# 概述

<PluginInfo name="workflow-manual" link="/handbook/workflow/plugins/manual"></PluginInfo>

当业务流程不能完全自动化决策时，可以通过人工节点，将部分决策权交给人工处理。

人工节点在执行到时会先中断整个流程的执行，生成对应用户的待办任务，在用户提交后根据所选的状态决定继续流程、继续等待还是终止流程。在需要进行流程审批等场景会非常有用。

## 安装

内置插件，无需安装。
