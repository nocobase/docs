# 通知：企业微信

<PluginInfo commercial="true" name="wecom"></PluginInfo>

## 介绍

**企业微信**插件支持应用向企业微信用户发送通知消息。

## 添加和配置企业微信认证器

首先需要在 NocoBase 上添加和配置一个企业微信认证器，参考 [用户认证 - 企业微信](./auth)，只有经过企业微信登录的系统用户，才可以通过企业微信接收系统通知。

## 工作流通知节点配置

选择配置好的企业微信通知渠道，支持三种消息类型：文本卡片，Markdown, 模版卡片。
