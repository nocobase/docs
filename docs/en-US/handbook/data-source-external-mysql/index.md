# 外部数据源 - MySQL

:::info
This feature is powered by plugin-data-source-external-mysql. 
:::

## 介绍

支持接入 MySQL 数据库，并将其中的数据表作为区块展示在界面上

## 安装

该插件为商业插件，需要通过插件管理器上传并激活插件
