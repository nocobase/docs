# Enregistrer un Enregistrement

### Introduction

L'opération de sauvegarde des données permet d'assigner des valeurs aux champs par le biais de l'assignation de champs, ce qui prend le pas sur toute valeur entrée dans le formulaire. Cette opération peut être associée à des flux de travail pour automatiser les processus de données de manière fluide.

![20240413214926](https://static-docs.nocobase.com/20240413214926.png)

### Éléments de Configuration de l'Opération

#### Assigner des Valeurs aux Champs

Lorsque l'assignation de champs est configurée, la valeur assignée remplacera toute saisie fournie dans le formulaire pour le même champ, assurant ainsi la cohérence et la priorité pour la valeur prédéfinie.
