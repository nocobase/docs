# **Enregistrer l'enregistrement**

Les opérations personnalisées d'**Enregistrement d'enregistrement** sont spécifiques aux blocs de formulaire, permettant de personnaliser le comportement de l'opération :

- **Bouton de modification**
- **Attribuer des valeurs aux champs** : Lorsque le bouton personnalisé est cliqué, les assignations de données actuelles des champs sont enregistrées.
- **Ignorer la validation des champs obligatoires**
- **Après la soumission réussie**
