# 1Panel 面板安装

## 0. 先决条件

⚡⚡ 请确保你已经安装了 [1Panel](https://1panel.cn/docs/) 面板

## 1. 搜索应用

1. 登录 1Panel 管理面板
2. 点击左侧菜单栏中的 **「应用商店」**
3. 在 **「开发工具」** 分类下找到 NocoBase 应用

## 2. 安装配置

点击 **「安装」** 按钮，进入安装配置页面，填写以下信息：

- **名称**：要创建的 NocoBase 应用的名称
- **版本**：选择 NocoBase 版本（推荐使用最新稳定版）
- **端口**：设置服务访问端口
- **数据库服务器**：选择已安装的数据库类型
  - 支持：MySQL / PostgreSQL / MariaDB
  - *如未安装数据库，请先在应用商店中安装*
- **数据库名**：NocoBase 使用的数据库名称
- **数据库用户名**：数据库连接用户名
- **数据库用户密码**：数据库连接密码
- **应用密钥**：用于加密和解密数据的密钥（可以使用默认值）
- **时区**：输入服务器所在时区（默认 Asia/Shanghai）
- **高级设置**：务必勾选端口外部访问

点击 **「确认」** 按钮开始安装，系统将自动：

- 下载 NocoBase Docker 镜像
- 创建并配置数据库
- 启动应用服务

## 3. 登录 NocoBase

1. 进入 **「已安装」** 应用列表
2. 找到 NocoBase 应用
3. 点击 **「跳转」** 按钮即可打开 NocoBase

提示：建议先在 **「面板设置」** 中配置 **「默认访问地址」**，以便直接跳转。

## 4. 管理 NocoBase

在 **「已安装」** 页面中，你可以对 NocoBase 应用执行以下操作：

- **服务控制**：启动 / 停止 / 重启
- **运维操作**：查看日志 / 备份与恢复 / 进入终端
- **应用管理**：重建应用 / 卸载应用 / 编辑应用
