# 外部数据源 - MySQL

<PluginInfo commercial="true" name="data-source-external-mysql"></PluginInfo>

## 介绍

使用外部的 MySQL 数据库作为数据源。目前支持的版本： MySQL >= 5.7

## 安装

该插件为商业插件，需要通过插件管理器上传并激活插件
