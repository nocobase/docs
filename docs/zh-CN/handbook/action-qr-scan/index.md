## 使用手册

### 生成二维码

1. 假如需要跳转的页面链接为： `https://localhost:13000/m/page/vyoiwa25jig`
2. 我们从手机页面提取 `/page/` 开始时的相对链接，生成二维码。
3. 新建markdown区块，填写如下：

```markdown
<qr-code value="/page/tr8r70ajpko" type="svg"></qr-code>
```

4. 新建“扫描二维码”操作，扫描并获跳转对应页面。

注：
- 扫描二维码操作，只支持系统内部相对链接，且需要从 `/page/` 开始生成
- 暂不支持外部页面。
