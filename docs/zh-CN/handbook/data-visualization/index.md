# 数据可视化

<PluginInfo name="data-visualization"></PluginInfo>

## 介绍

NocoBase 的数据可视化插件提供了可视化的数据检索功能和丰富的图表组件，基于系统 Collection 数据，用户可以快速建立可视化面板，灵活组织图表，进行业务数据分析。

![](https://static-docs.nocobase.com/51be43d5400d6294f6c20d11009f23c4.png)

## 安装

内置插件，无需单独安装。
