### 1个度量，1个轴维度，一个分类维度

配置1个度量，2个维度。其中1个维度作为 X 轴字段 (X field), 另一个作为分类字段 (Series field).

![](https://static-docs.nocobase.com/202410101113944.png)

## 配置项

| 配置项   | 说明                                   |
| -------- | -------------------------------------- |
| X 轴字段 | X 轴维度字段                           |
| 分类字段 | 分类维度字段                           |
| 尺寸     | 设置图表的大小，固定宽高比或者固定高度 |
| 平滑曲线 | 是否平滑曲线                           |
| 是否堆叠 | 是否叠加数据值                         |
