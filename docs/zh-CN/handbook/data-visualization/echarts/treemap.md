## 配置项

| 配置项       | 说明                                   |
| ------------ | -------------------------------------- |
| 尺寸         | 设置图表的大小，固定宽高比或者固定高度 |
| 浅色模式主题 | 需要在系统为浅色模式时选择才能预览     |
| 深色模式主题 | 需要在系统为深色模式时选择才能预览     |
| 标签类型     | 控制标签是否显示以及显示类型           |
