# 提交至工作流

## 介绍
当新增或更新数据时，用户可以选择通过配置按钮的“绑定工作流”选项，或者直接使用“提交至工作流”按钮，将数据提交至绑定的工作流。一旦绑定了工作流，点击按钮将触发相应的工作流程。

相比较于“绑定工作流”选项，“提交至工作流”按钮可以直接将数据提交到工作流处理，而其他操作则会在数据提交成功后触发，如按钮原有的操作逻辑处理失败，则不会触发该事件。
