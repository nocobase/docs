# 子表单（弹窗）

## 介绍

对于关系层级复杂、字段众多的关系表，子表单布局可能难以有效呈现数据的主次关系。子表单（弹窗）则针对这种场景，将关系表单以弹窗形式显示，同时支持在子表格中使用。
