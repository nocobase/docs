# 子表单

## 介绍

子表单适用于新建关系数据并关联关系数据的场景，在处理深层次的关系数据时效果显著，与数据选择器和下拉选择器相比，子表单在当前页面上直接维护关联信息，省去了频繁打开弹窗的步骤，减少了用户的点击次数。通过嵌套表单的形式清晰展示多层级的关系数据，直接在子表单中新建关联数据并关联到主表（随主表一起提交），为用户带来更流畅和高效的数据关联和管理体验。
