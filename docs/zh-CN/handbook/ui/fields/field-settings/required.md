# 必填

## 介绍

必填是表单验证的常用规则，可以通过静态设置直接在字段属性中开启必填，还可以表单的联动规则动态设置字段必填。
## 静态设置

直接设置字段为必填项。这种方式是最简单直接的，适用于那些始终需要用户填写的字段，如用户名、密码等

## 动态设置(条件必填)

通过联动规则按条件设置必填

如下示例： 当状态为已发货时发货日期字段必填
