# 批量编辑表单字段#

## 介绍

批量编辑表单是批量操作特有的表单。

![20240422151115](https://static-docs.nocobase.com/20240422151115.png)
## 更新逻辑

- 不更新：这个字段就不会更新；
- 修改为：这个字段为必填；
- 清空：提交后就会清空这个字段的值；
