# 设置排序规则

## 介绍

设置排序规则可以为数据区块的一个或多个字段定义默认排序顺序。通过选择升序或降序排列，可以自定义数据的呈现方式并提高信息检索效率。有助于揭示数据的关键趋势，使数据分析更加直观易懂
