## `x-designer` 和 `x-initializer` 的使用场景

`x-decorator` 或 `x-component` 是以下组件时，`x-designer` 生效：

- BlockItem
- CardItem
- FormItem
- Table.Column
- Tabs.TabPane

`x-decorator` 或 `x-component` 是以下组件时，`x-initializer` 生效：

- ActionBar
- BlockItem
- CardItem
- FormItem
- Grid
- Table
- Tabs
