## NocoBaseとは

NocoBaseは、非常に拡張性の高いオープンソースのノーコード開発プラットフォームです。  
数年の時間や数百万の資金を投資することなく、数分でNocoBaseを展開し、すぐにプライベートで制御可能な、極めて拡張性の高いノーコード開発プラットフォームを手に入れることができます。
