### 1 メジャー, 1 ディメンション

## 設定オプション

| オプション             | 説明                                      |
| ------------------ | ------------------------------------------------ |
| サイズ               | チャートのサイズを設定します。固定アスペクト比または固定高さを選択できます |
| ライトモードテーマ   | システムがライトモードの場合にプレビューするためにこのオプションを選択します |
| ダークモードテーマ    | システムがダークモードの場合にプレビューするためにこのオプションを選択します |
| 形状              | チャートの形状を制御します                 |
