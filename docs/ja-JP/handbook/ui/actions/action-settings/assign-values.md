# フィールドの値設定

## 概要

フィールドの値設定とは、操作を実行する前にフィールドの値をあらかじめ設定し、レコードの更新や作成時にこれらのデータを自動的に入力することを指します。手動でデータを入力する必要がなく、例えば、注文ステータスを一括更新したり、注文が配達された際に配達時間を現在の時間に設定する場面で利用されます。
