# 表示

## 概要

表示操作は行ボタンに属し、現在のレコードをコンテキストとして使用し、ポップアップ形式で表示されます。ユーザーはポップアップ内で構成ブロックをカスタマイズすることができ、インターフェースの柔軟性を高めます。
