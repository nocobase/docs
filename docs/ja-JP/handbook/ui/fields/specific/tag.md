# タグ

## 紹介

タグは、リレーションフィールドの閲覧モードにおける表示コンポーネントであり、タイトルフィールドとカラーフィールドの設定が必要です。

## 使用説明

### テーブルでの使用

注文表には、多対一のリレーションフィールド「タグ」があります。

タグ表には「タグ名」と「タグカラー」の2つのフィールドがあります。
