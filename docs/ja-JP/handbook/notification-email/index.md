# 通知：電子メール

<PluginInfo name="notification-email"></PluginInfo>

## 概要

電子メールチャネルを通じて通知を送信する機能です。現在はSMTPプロトコルのみ対応しています。

## インストール

このプラグインはシステムにプリインストールされていますが、使用するためには有効化が必要です。
