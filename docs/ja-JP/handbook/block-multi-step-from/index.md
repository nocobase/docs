# ステップ式フォーム

<PluginInfo commercial="true" name="block-multi-step-from"></PluginInfo>

## はじめに
ステップ式フォームは、データ入力および編集のインターフェースを構築するためのブロックであり、フォームブロックの機能を基にステップ構成をサポートします。
