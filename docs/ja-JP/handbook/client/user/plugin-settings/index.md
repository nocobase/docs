# プラグイン管理センター

管理センターは、ユーザーにシステム情報や業務関連の設定を一元管理・構成するためのプラットフォームを提供します。ユーザーは、特定のビジネスニーズや個人の好みに応じて、システム、データテーブルモデリング、ワークフロー設定、プラグイン設定をカスタマイズできます。

## プラグイン管理センターの構成

1. 先頭にあるクイック設定項目
2. すべてのアクティブなプラグイン設定のリスト
3. プラグインのタブ
4. メインコンテンツエリア
