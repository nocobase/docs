# 使用マニュアル

NocoBaseのクライアントインターフェースを提供する組み込みコアプラグインの一つです。クライアントにはAdminLayoutレイアウトが組み込まれており、主な構成要素は以下の通りです。

## プラグインマネージャー

プラグインの管理が可能で、追加、アクティベート、無効化、および削除を行えます。

## プラグイン管理センター
