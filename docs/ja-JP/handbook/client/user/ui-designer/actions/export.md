# エクスポート

エクスポート操作はプラグインの拡張を通じて実現されており、ブロックに関連付けられたデータテーブルのすべてのデータをバッチエクスポートすることができます。この機能では、エクスポートするフィールドのタイトルを設定することもサポートされています。
