### **Просмотр**

#### **Обзор**

Операция «Просмотр» — это кнопка уровня строки, которая использует текущую запись в качестве контекста. Открывается во всплывающем окне и позволяет пользователям настраивать блоки содержимого, повышая гибкость интерфейса.
